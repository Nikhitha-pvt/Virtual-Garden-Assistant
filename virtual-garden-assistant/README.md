# Virtual Garden Assistant

A 3D garden planning tool with drag-and-drop functionality, allowing users to design and visualize their garden spaces before implementing them in real life.

## Features

- 3D and 2D view of your garden design
- Drag and drop elements (trees, plants, furniture, etc.)
- Manual grid drawing for precise garden planning
- Variety of elements: 10+ plants, ponds, furniture, houses and more
- Save, load, and share your garden designs
- Undo/Redo functionality
- Clean and intuitive user interface

## Technologies Used

- HTML5, CSS3, and JavaScript
- Three.js for 3D rendering
- Bootstrap for responsive design
- React for component-based UI
- MySQL and PHP for backend storage (in progress)

## Getting Started

### Prerequisites

- Bun 1.0.0 or later
- Node.js 18.0.0 or later

### Installation

1. Clone the repository
   ```
   git clone https://github.com/Nikhitha-pvt/Virtual-Garden-Assistant.git
   cd virtual-garden-assistant
   ```

2. Install dependencies
   ```
   bun install
   ```

3. Run the development server
   ```
   bun run dev
   ```

4. Open your browser and navigate to http://localhost:3000

## Building for Production

```
bun run build
```

The built files will be in the `dist` directory.

## Usage Guide

1. Use the left sidebar to select element categories
2. Click on or drag elements onto the garden canvas
3. Use the properties panel on the right to customize elements
4. Toggle between 2D and 3D views with the view controls
5. Use the top toolbar for file operations and history controls
6. Save your design and share it with others
